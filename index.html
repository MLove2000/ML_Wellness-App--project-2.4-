Summary

I created an App for improving mental health. Specifically retraining your brain to think more positively. The purpose of the app is to encourage self love, confidence, and positive thinking. The user does this by participating in daily activities in the app. the app tracks progress through mood and timing on the activity. Each activity is ten minutes. The users mood is asked before and after the activity.

Target User

My target audience would be anyone who is struggling with negative thoughts, depression, or needs more confidence in themselves. It's easy to use so you can use it whenever and where ever. There's no age limit since anyone can struggle with these problems.

Process

During my researching process, I downloaded a couple apps to help see what I wanted to use. I wanted to make sure the app would be easy to use and understandable to new users. Some apps I used to achieve this were Headspace: Mindful Meditation and InnerHour: Self-care therapy.

User Interviews

I got some good useful points in the User Interviews. Most were small details but very helpful. It helped me narrow a lot of things down as well. I had a lot of ideas and the interview helped guide me into one direction.

Challenges

Some challenges I had during this project were getting too carried away, staying on a specific track, space, and changes. I made a lot of changes because I had a set goal but wasn't sure how to get exactly where I wanted to be. The concept itself was a bit of a challenge. I was able to get past these with help from the user interviews and researching other wellness apps.

Successes

Some successes I had during this project were making it easy to use, consistent, and understandable. Testability seemed really easy to use and there weren't many questions but a lot of helpful suggestions. I had a ton of ideas to start from as well so I already had a good place to start from.
